# Vue Todo

A Simple Todo app built with [vue.js](https://vuejs.org/)

![alt text](screenshot.png "Screenshot")

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Lints and fixes files

```
yarn run lint
```
